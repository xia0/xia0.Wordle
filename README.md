# xia0.Wordle
![Wordle game in Titanfall 2 Northstar](https://i.imgur.com/sWzxxy0.png)
|ConVar|Default|Description|
|---|---|---|
|wordle_blank_character|#|`char` What character to use to indicate a blank square. Unicode characters do not seem to work.|
|wordle_guesses|6|`int` How many guesses players should get. Do not decrease below 3 as the keyboard will no longer display correctly.|
|wordle_share_at_map_end|1|`bool` Share everybody's results during postmatch.|
